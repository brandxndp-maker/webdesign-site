Кто такой веб-дизайнер?

Веб-дизайнер — это специалист, который отвечает за внешний вид и ощущение сайта. Он создает макеты, продумывает расположение элементов и заботится о том, чтобы пользователю было удобно.

Основные термины

  • UI (User Interface) — визуальное оформление интерфейса.
  • UX (User Experience) — удобство и логика взаимодействия.
  • Типографика — искусство оформления текста.
  • Модульная сетка — каркас для расположения блоков.
  • Адаптивность — корректное отображение на всех устройствах.

Главный инструмент для дизайна интерфейсов. Работает в браузере.
